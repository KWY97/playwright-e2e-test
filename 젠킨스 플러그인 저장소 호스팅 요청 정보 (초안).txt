젠킨스 플러그인 저장소 호스팅 요청 정보 (초안)

1. 플러그인 이름:
Display Name: Playwright E2E Test Plugin
Artifact ID: playwright-e2e-test

2. 현재 저장소 URL: 
https://github.com/KWY97/playwright-e2e-test

3. 플러그인에 대한 간략하고 명확한 설명:
이 플러그인은 Jenkins 환경에서 Playwright MCP를 활용하여 대화형 E2E(End-to-End) 자동화 테스트를 실행하고 관리할 수 있도록 지원합니다.
LLM(Large Language Model) 기반의 대화형 인터페이스를 통해 테스트 케이스 정의, 실행, 결과 분석을 자동화하는 기능을 제공합니다.
주요 기능에는 테스트 설정 (테스트 케이스/스크린샷 경로 지정, API Key 저장 등), 대화형 Instruction 설정 (응답 언어 등), 그리고 Jenkins CI/CD 파이프라인 내 테스트 실행 및 결과 연동 기능이 포함됩니다.

4. 플러그인의 목적 및 중요성:
목적: 기존의 스크립트 기반 E2E 자동화 테스트가 가지는 유연성 부족과 유지보수의 어려움을 개선하고자 합니다. 최신 LLM 및 MCP 기술을 Jenkins 파이프라인에 통합하여, 사용자가 보다 직관적이고 효율적인 방식으로 E2E 테스트를 수행할 수 있도록 하는 것이 목표입니다.

중요성: 젠킨스 사용자들이 복잡한 외부 연동 없이도 익숙한 CI/CD 환경 내에서 최첨단 대화형 자동화 테스트 기술을 쉽게 활용할 수 있게 함으로써, 테스트 자동화의 접근성과 효율성을 크게 높일 수 있습니다. 이는 젠킨스 에코시스템의 테스트 자동화 기능을 확장하는 데 기여할 것입니다.

차별점: Playwright의 강력함과 LLM/MCP의 대화형 상호작용을 Jenkins에 통합한 점이 주요 차별점입니다.

5. 사전 준비 상태:
플러그인 코드는 개발 완료되었으며, Jenkins Plugin 구조에 맞춰 .hpi 파일 생성이 가능합니다.
pom.xml 설정 및 MIT 오픈소스 라이선스 적용(파일 및 pom.xml 명시)은 완료되었습니다.
플러그인 코드는 현재 GitHub 저장소에 푸시되어 있습니다.
사용자 문서(README.md) 작성 및 상세 테스트는 현재 진행 중입니다. 공식 호스팅 요청 제출 전까지 완료될 예정입니다. (실제 요청 시에는 이 부분이 완료되었다고 명확히 기재해야 합니다.)
